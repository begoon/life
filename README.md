# The game of Life in Svelte/Javascript

Live demo: [https://demin.ws/life/](https://demin.ws/life/)

## Prerequisites

- [Node.js](https://nodejs.org/en/)

## Installation

```bash
npm install
```

## Local build and run

```bash
npm run dev
```

## Build for deployment

```bash
npm run build
```

## Preview the production build

```bash
npm run preview
```
